# Angular-Vue-Svelte

Ett skolprojekt för att testa tre populära frontend-ramverk. Jag valde att göra en json-server med API för att hämta hundar och lägga till/ta bort från favoriter. Denna uppgift är gjort i Angular, Vue och Svelte. För att testa projektet, börja med att klona ner repot eller hämta det som ZIP fil och öppna i din kodeditor. 

### Starta projektet i angular
* Kör: npm install
* Öppna terminalen och navigera till json-server genom att skriva: cd json-server <br>
* Skriv sedan: npx json-server --watch db.json --port 3000 (för att köra json-servern) <br>
* Öppna en ny flik i terminalen och skriv: cd angular <br>
* Skriv sedan: ng serve (för att starta servern för angular) <br>
* Kopiera servern till din webbläsare (vanligtvis http://localhost:4200)
